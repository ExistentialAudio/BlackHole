BlackHole: Virtual Audio Driver

BlackHole is a modern open source virtual audio driver for macOS that allows applications to pass audio to other applications with zero additional latency.

Features

  • 2, 16, 64, 128, and 256 channel versions available
  • 8kHz, 16kHz, 44.1kHz, 48kHz, 88.2kHz, 96kHz, 176.4kHz, 192kHz, 352.8kHz, 384kHz, 705.6kHz and 768kHz sample rates
  • No driver latency
  • Runs on macOS 10.9+
  • Built for Intel and Apple Silicon
